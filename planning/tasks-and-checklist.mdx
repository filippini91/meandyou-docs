## Adding recommended tasks

Not sure where to start? Select **Add recommended tasks** to bring in a curated set of common wedding to-dos, with due dates calculated automatically from your wedding date.

<Note>
  Recommended tasks need a wedding date to calculate due dates. If you haven't set one yet, you'll be prompted to add it first — see [Overview](/planning/overview).
</Note>

## Adding your own tasks

<Steps>
  <Step title="Select New task">
    Give it a name and, optionally, a due date, tags, and an assignee.
  </Step>
  <Step title="Save">
    Your task appears in the list.
  </Step>
  <Step title="Check it off when done">
    Select the checkbox next to a task to mark it complete.
  </Step>
</Steps>
