**File storage** is a shared folder for anything you don't want to lose in your email — venue contracts, invoices, inspiration PDFs, and more.

<Frame>
  ![20 File Storage](/images/20_fileStorage.jpg)
</Frame>

{/* TODO(screenshot): File storage list with 3-4 sample files (e.g. a contract, an invoice) showing who uploaded each */}

<Steps>
  <Step title="Select Upload">
    Choose one or more files from your device.
  </Step>
  <Step title="Find it later">
    Use search, or browse the list — each file shows who uploaded it and when.
  </Step>
  <Step title="Download or delete">
    Select a file to download it, rename it, or remove it if you no longer need it.
  </Step>
</Steps>

<Tip>
  Anyone you've invited to co-plan with you can see and add files here — it's a shared space, not a personal one.
</Tip>
